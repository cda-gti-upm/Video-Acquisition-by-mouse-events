* Use the same library as for mouse events.
* R1 remote control

  * VK_VOLUME_UP	175	Windows 2000: Volume Up key
  * VK_MEDIA_PREV_TRACK	177	Windows 2000: Previous Track key
  * VK_MEDIA_STOP	178	Windows 2000: Stop Media key
